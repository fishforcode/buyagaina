## 🗂️ 数据库结构

### 已实现集合

#### `locations` 集合
```javascript
{
  _id: string,
  name: string,           // 空间名称
  icon: string,           // 空间图标
  remark: string,         // 备注
  createTime: timestamp,  // 创建时间
  createdBy: string       // 创建者openid
}
```

#### `items` 集合
```javascript
{
  _id: string,
  name: string,           // 物品名称
  quantity: number,       // 数量
  expiryDate: timestamp,  // 保质期
  remark: string,         // 备注
  locationId: string,     // 所属空间ID（外键）
  createTime: timestamp,   // 创建时间
  createTimeStr: string,  // 创建时间字符串
  status: string          // 状态
}
```

### 待实现集合

#### `user_settings` 集合（智能提醒系统）
```javascript
{
  _id: string,
  _openid: string,           // 用户openid
  reminderDays: number,      // 提前提醒天数（默认3）
  reminderEnabled: boolean,  // 是否开启提醒（默认true）
  updateTime: timestamp
}
```

#### `families` 集合（规划中）
```javascript
{
  _id: string,
  name: string,           // 家庭名称
  creatorId: string,      // 创建者ID
  createTime: timestamp
}
```

#### `family_members` 集合（规划中）
```javascript
{
  _id: string,
  familyId: string,       // 家庭ID（外键）
  userId: string,         // 用户ID
  role: string,           // 角色（owner/admin/member）
  joinTime: timestamp
}
```

### v1.0.1 详细变更（2026-03-31）
- **体验优化**
  - 数量输入框允许清空，输入更灵活
  - 空间详情支持100+物品，统计更准确
  - 删除空间时自动级联删除关联物品
- **Bug修复**
  - 修复提醒逻辑：手动点击按钮始终发送，不受每日限制
  - 修复结果处理：正确区分"无临期物品"和"今天已提醒"
  - 清理云函数调试日志

### v1.2.0 详细变更（2026-06-29）
- **UI视觉系统全面升级**
  - 采用莫兰迪色系设计，打造高级、清新、统一的视觉风格
  - 核心色彩体系：柔和蓝绿主色(#7C9A92)、淡雅灰绿辅色(#B8C4BB)、温暖米白背景(#F5F5F3)
  - 统一视觉语言：细字体(weight: 300-400)、大圆角(16-48rpx)、轻微阴影、宽松间距
  - 全局样式系统：创建app.wxss全局样式文件，定义色彩、字体、间距、圆角、阴影等设计规范
- **页面样式更新**
  - 首页(index.wxss)：清新的空间列表、优化的导航栏和快捷功能栏
  - 空间详情页(location.wxss)：优雅的统计卡片、精致的物品列表展示
  - 物品详情页(item_detail.wxss)：Hero区域设计、信息卡片布局、时间线展示
  - 创建物品页(create_item.wxss)：现代表单设计、优化的快捷选择区、单位选择弹窗
  - 创建空间页(create_location.wxss)：简洁的输入界面、图标选择器优化
- **设计理念**
  - 摒弃过于鲜艳的颜色，采用低饱和度色彩
  - 卡片式布局，清晰的信息层级
  - 轻盈优雅的交互体验

### v1.0.0 详细变更
- 空间管理（创建、重命名、删除）
- 物品管理（添加、编辑、删除、消耗）

- 全新UI设计（紫蓝渐变、卡片式布局）
