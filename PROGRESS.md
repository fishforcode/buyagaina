## 🗂️ 数据库结构

### 已实现集合

#### `locations` 集合
```javascript
{
  _id: string,
  name: string,           // 空间名称
  icon: string,           // 空间图标
  remark: string,         // 备注
  createTime: timestamp,  // 创建时间
  createdBy: string       // 创建者openid
}
```

#### `items` 集合
```javascript
{
  _id: string,
  name: string,           // 物品名称
  quantity: number,       // 数量
  expiryDate: timestamp,  // 保质期
  remark: string,         // 备注
  locationId: string,     // 所属空间ID（外键）
  createTime: timestamp,   // 创建时间
  createTimeStr: string,  // 创建时间字符串
  status: string          // 状态
}
```

### 待实现集合

#### `user_settings` 集合（智能提醒系统）
```javascript
{
  _id: string,
  _openid: string,           // 用户openid
  reminderDays: number,      // 提前提醒天数（默认3）
  reminderEnabled: boolean,  // 是否开启提醒（默认true）
  updateTime: timestamp
}
```

#### `families` 集合（规划中）
```javascript
{
  _id: string,
  name: string,           // 家庭名称
  creatorId: string,      // 创建者ID
  createTime: timestamp
}
```

#### `family_members` 集合（规划中）
```javascript
{
  _id: string,
  familyId: string,       // 家庭ID（外键）
  userId: string,         // 用户ID
  role: string,           // 角色（owner/admin/member）
  joinTime: timestamp
}
```
