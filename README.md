# 又买啊

一个帮助记录物品保质期、提醒临期物品的小程序。

## 功能

- 📦 创建空间（冰箱、储物柜等）管理物品
- 📅 记录物品保质期和数量
- 🔔 临期提醒（每天自动检查）

## 技术栈

- 微信小程序
- 微信云开发（云函数 + 数据库）
- 订阅消息

## 快速开始

1. 克隆项目
2. 复制 `miniprogram/config.example.js` 为 `config.js`
3. 填入你自己的云环境ID和模板ID
4. 在微信开发者工具中导入项目
5. 部署云函数

## 配置说明

见 `miniprogram/config.example.js` 注释
